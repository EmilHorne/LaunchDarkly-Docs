## Overview

This topic links to Heap's documentation for its LaunchDarkly integration. Heap is a customer behavioral data analytics tool. You can send user key and feature flags to Heap as user and event properties.

Heap manages this integration. To learn more, read Heap's [LaunchDarkly Integration documentation](https://help.heap.io/integrations/testing-and-personalization/launchdarkly/).
