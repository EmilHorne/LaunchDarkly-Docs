## Overview

This topic links to the documentation for the LaunchDarkly Ansible integration. Ansible is an automation tool you can configure to manage some tasks in your LaunchDarkly account, such as creating projects and flags or syncing data across environments.

To learn more, read the [Ansible integration documentation](https://launchdarkly-labs.github.io/ansible-launchdarkly-collection/).
