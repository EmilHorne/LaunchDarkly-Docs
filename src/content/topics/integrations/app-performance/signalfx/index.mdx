## Overview

This topic explains the LaunchDarkly SignalFx integrations available, including events and triggers.

[SignalFx](https://www.signalfx.com/) is an [application performance management (APM)](https://en.m.wikipedia.org/wiki/Application_performance_management) tool. It allows customers to monitor the operational health of their application and infrastructure.

## Events

The LaunchDarkly SignalFx events integration exposes flag change data to SignalFx, which can detect and alert engineers when their applications are trending in an unstable direction. To learn how to set up this integration, read [Using the SignalFx events integration](/integrations/signalfx/events).

## Triggers

 LaunchDarkly supports using flag triggers with SignalFx. For example, you can create an alert in SignalFx that toggles a flag on or off if a performance metric drops below a certain threshold. To learn how, read [Using flag triggers with SignalFx](/integrations/signalfx/triggers).
